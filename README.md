# Furniture Image Classification using CNN

A Convolutional Neural Network (CNN) project for classifying furniture images into different categories.

## Files

* `cnn.py` – Model training
* `predict.py` – Image prediction
* `capture.py` – Image capture utility
* `test.jpg` – Sample test image

## Technologies

* Python
* TensorFlow/Keras
* OpenCV
* NumPy

## Run

Train the model:

```bash
python cnn.py
```

Predict an image:

```bash
python predict.py
```

## Note

The dataset and trained model are not included in this repository due to their large size.
